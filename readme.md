Modelo Relacional de Películas

Este proyecto consiste en el diseño e implementación de una base de datos relacional en MySQL a partir de datos de películas en formato JSON.

Se ha creado un modelo normalizado con tres tablas: peliculas, generos y subtitulos, utilizando claves primarias y foráneas para garantizar la integridad referencial.

Relaciones implementadas:

Una película pertenece a un género (N:1).

Una película puede tener varios subtítulos (1:N).

El archivo evaluacion-final-salika-iciar contiene la creación de tablas y definición de relaciones.

El archivo evaluacion-final-iciar incluye la inserción de datos y la resolución de consultas SQL.

Se aplican conceptos como:

JOIN

GROUP BY

funciones de agregación (COUNT, AVG, MAX)

subconsultas

consultas con LIKE

El objetivo es demostrar comprensión del diseño relacional y consultas SQL intermedias.
